Quando seria mais adequado utilizar CSS Grid em vez de display flex?

Resposta 1

No caso do uso de mais de um container flex para organizar elementos em um layout, aplicar diversas propriedades nos elementos filhos para ter maior controle do layout, haver a existência de elementos diretos que ocupam tanto linhas quanto colunas, e, por último, caso apenas elementos se encaixem dentro de um layout, o ideal é usar grid.

Resposta 2

Quando o desenvolvedor quer que o conteúdo se baseie num layout e não o contrário. Quando se necessário, estruturar e delimitar a página.

Dê exemplos de layouts onde o Grid seria a melhor escolha.
